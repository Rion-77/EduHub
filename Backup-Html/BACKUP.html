What is a key property of a Directed Acyclic Graph (DAG) that makes it useful for scheduling tasks?
A) It contains at least one directed cycle
B) It has no directed cycles, allowing topological ordering
C) Every node has exactly two outgoing edges
D) It can only represent linear workflows
AI explanation preview

A DAG has no directed cycles, which guarantees a topological order — essential for scheduling tasks with dependencies. Option B is correct.

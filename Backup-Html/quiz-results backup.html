A
Bubble Sort ✗ Your answer
C
Topological Sort ✓ Correct
AI Explanation

Topological Sort is designed specifically for DAGs. It produces a linear ordering of nodes such that for every directed edge from u → v, node u comes before v. This is essential in dependency resolution — task scheduling, build systems, and course prerequisites. Bubble Sort is a general-purpose comparison sort with no awareness of graph structure.

AI Explanation

A DAG does not need to be fully connected. It is perfectly valid for a DAG to have disconnected components. The only mandatory property is that it is directed (edges have a direction) and acyclic (following edges never leads back to a previously visited node). Connectivity is a separate graph property entirely.
